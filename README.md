## 🚀 Déploiement GitHub Pages (GRATUIT)

### 1. Créer un dépôt GitHub
```bash
git init
git add .
git commit -m "ALGERIA MEGASYS v5.0 ULTRA — Initial Deploy"
git remote add origin https://github.com/VOTRE_USERNAME/algeria-megasys.git
git push -u origin main
```

### 2. Activer GitHub Pages
1. Aller dans **Settings** → **Pages**
2. Source : **Deploy from a branch**
3. Branch : `main` → `/ (root)`
4. Cliquer **Save**
5. URL : `https://VOTRE_USERNAME.github.io/algeria-megasys/`

### 3. Déploiement automatique (CI/CD)
Le fichier `.github/workflows/deploy.yml` est préconfiguré.  
Chaque `git push` redéploie automatiquement.
